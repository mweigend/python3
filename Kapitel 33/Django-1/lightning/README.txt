In diesem Ordner ist die erste Version des Django-Projekts "Ideenspeicher" die bis einschließlich Abschnitt 33.11.2  beschrieben worden ist. 
Das Projekt ist noch unvollständig. Es fehlt noch die Möglichkeit neue Fragen und neue Ideen einzugeben. 

So starten Sie den Server: 
Geben Sie auf der Kommandozeile in diesem Verzeichnis den folgenden Befehl ein:
python manage.py runserver


Öffnen Sie dann einen Browser und geben Sie folgende Adresse ein:

http://localhost:8000/ideas
