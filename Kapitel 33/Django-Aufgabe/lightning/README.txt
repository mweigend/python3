Kapitel 33

In diesem Ordner ist die Lösung der Aufgabe zum Django-Projekt "Ideenspeicher", die in Kapitel 33 beschrieben worden ist.  

So starten Sie den Server: 
Geben Sie auf der Kommandozeile in diesem Verzeichnis den folgenden Befehl ein:
python manage.py runserver


Öffnen Sie dann einen Browser und geben Sie folgenden URL ein:
http://localhost:8000/ideas

Jetzt können Sie weitere Fragen und Ideen eingeben.
Ein gültiges Schlüsselwort für die Definition einer neuen Frage ist magoba.

Um das Projekt zu verwalten (und z.B. weitere Schlüsselwörter einzugeben), verwneden Sie folgenden URL: 
http://localhost:8000/admin
